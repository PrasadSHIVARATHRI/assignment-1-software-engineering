Use command Prompt to Compile and run the java program. 

Just follow the below steps to get your output.

Step 1: Open Command prompt

Step 2: Now enter into the directory where your source file resides. (Use cd command to enter into the directory)

Step 3: Once you entered into the source file directory. Now you compile your java file.

Step 4: Type "javac Filename.java".

Step 5: Ensure there is no error after compiling. Now run the java program.

Step 6: Type "Java Filename.Java". Now u will get the output
